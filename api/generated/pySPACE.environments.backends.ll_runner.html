ll_runner¶

Module: environments.backends.ll_runner¶

Executable script for Loadleveler to start processes

A simple helper script that unpickles, executes and deletes a given process. Afterwards the LoadLevelerComHandler of LoadLeveler Backend is informed that execution is finished.
